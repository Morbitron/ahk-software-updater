Hello again.

If that works for you I'm glad, but it seems unnecessarily complicated. Wouldn't this also work for you?

Code: [Select all] [Expand] [Download] GeSHi © Codebox Plus

URL := "http://mtgjson.com/json/"

URLDownloadToFile, %URL%, %A_Temp%\dir.html ; saves the webpage to a temp file
Loop, Read, %A_Temp%\dir.html ; reads the temp file line by line
{
	StringSplit, FileInfo, A_LoopReadLine, `" ; quotes (") surround the filenames in the web page
	If ((FileInfo0 == 3) and (FileInfo2 != "../") and (FileInfo3 != ">")) ; filters out undesired lines
	{
		; Now do something with FileInfo2 (each linked filename)
		MsgBox %URL%%FileInfo2%
	}
}
FileDelete, %A_Temp%\dir.html ; deletes the temp file

With a bit more work you could also get the date and the filesize (they are in FileInfo3), and without additional downloads.

If the structure of the file doesn't change why you do not keep the new downloaded file and compare it to the old one.
And after you can do what you want with it.
Exemple.

Code: [Select all] [Expand] [Download] GeSHi © Codebox Plus

URL := "http://mtgjson.com/json/"
URLDownloadToFile, %URL%, %A_Temp%\new.html

FileRead, NEW, %A_Temp%\new.html
FileRead, OLD, %A_Temp%\old.html
If ( NEW = OLD ) ;the file is unchanged
}
	MsgBox SAME
}
else ;keep the new file
{
	FileMove, %A_ScriptDir%\new.html, %A_ScriptDir%\old.html, 1
}
